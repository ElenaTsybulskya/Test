# Инструкция по установке Postman

## Введение

Postman — это платформа для работы с API, которая позволяет разрабатывать, тестировать и документировать API.

## Системные требования

- **ОС:** Windows 10/11, macOS 10.12+, Ubuntu 16.04+
- **Память:** 4 GB RAM (рекомендуется 8 GB)
- **Место на диске:** 500 MB свободного места

## Шаг 1: Скачивание установщика

1. Перейдите на [официальный сайт Postman](https://www.postman.com/downloads/)
2. Нажмите кнопку **`Download for Windows`**
3. Дождитесь завершения загрузки файла `Postman-win64-version-Setup.exe`

> **Примечание:** Версия для Windows автоматически определяется как 64-bit

## Шаг 2: Установка Postman

### Вариант A: Стандартная установка
1. **Запустите** скачанный файл `Postman-win64-Setup.exe`
2. **Разрешите** приложению вносить изменения на вашем устройстве
3. Дождитесь завершения автоматической установки
4. Postman запустится автоматически после установки

### Вариант B: Установка с выбором пути
1. Запустите установщик
2. Нажмите **`Cancel`** при появлении окна автоматической установки
3. Распакуйте файлы в нужную директорию
4. Запустите `Postman.exe` из папки установки

## Шаг 3: Первоначальная настройка

### Создание учетной записи
1. **Запустите** Postman
2. Нажмите **`Create Account`** для создания аккаунта
3. Заполните данные:
   - Email адрес
   - Имя пользователя
   - Пароль
4. Подтвердите email через письмо на почте

> **Альтернатива:** Можно использовать кнопку **`Skip and go to the app`** для начала работы без аккаунта

### Настройка рабочего пространства
1. Выберите тип рабочего пространства:
   - **Personal** — для личного использования
   - **Team** — для командной работы
   - **Public** — для открытых проектов
2. Нажмите **`Continue`**

## Шаг 4: Проверка установки

### Тестовый запрос
1. Создайте новый запрос (**`File → New → HTTP Request`**)
2. Введите URL: `https://jsonplaceholder.typicode.com/posts/1`
3. Нажмите кнопку **`Send`**
4. Убедитесь, что получили ответ со статусом `200 OK`

```json
{
  "userId": 1,
  "id": 1,
  "title": "sunt aut facere...",
  "body": "quia et suscipit..."
}
```
## Шаг 5: Дополнительные настройки (опционально)

### Настройка прокси

1. File → Settings
2. Вкладка Proxy
3. Настройте параметры прокси при необходимости

### Настройка темной темы

1. File → Settings
2. Вкладка Theme
3. Выберите Dark для темной темы

### Возможные проблемы и решения

## Проблема: *Установщик не запускается*

### *Решение:*
- Проверьте антивирус
- Скачайте установщик заново
- Запустите от имени администратора

### Проблема: *Ошибка сети*

### *Решение:*

- Проверьте подключение к интернету
- Настройте прокси в Settings
- Отключите VPN на время установки


## Проблема: *Не запускается после установки*

### *Решение:*

- Перезагрузите компьютер
- Переустановите Postman
- Проверьте системные требования
